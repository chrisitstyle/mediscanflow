# MediScanFlow

- MediScanFlow is a distributed medical image analysis platform demonstrating asynchronous AI inference workflows with Spring Boot, FastAPI, RabbitMQ, MinIO, PostgreSQL and Next.js.

- The system allows users to upload medical scan images, queue AI analysis jobs, process them asynchronously using a Python inference worker and review generated results through a web interface.

- This project is built for educational and portfolio purposes. It is not intended for real medical diagnosis.
